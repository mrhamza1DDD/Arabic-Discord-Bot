# Arabic Discord Bot

هذا المستودع يحتوي على بوت ديسكورد بسيط باللغة العربية مبني باستخدام discord.js.

الأوامر المتوفرة:
- /ping — يختبر استجابة البوت ويعرض التأخير (بالعربية)
- /randomjoke — يعرض نكتة عشوائية باللغة العربية

متطلبات:
- Node.js 18 أو أحدث
- متغيرات بيئية: DISCORD_TOKEN و CLIENT_ID

تشغيل محلياً:
1. انسخ المتغيرات البيئية إلى ملف `.env` (اختياري):

DISCORD_TOKEN=توكن_البوت
CLIENT_ID=معرف_التطبيق

2. تثبيت الحزم:

npm install

3. تشغيل البوت:

npm start

النشر على Render:
1. أنشئ خدمة جديدة من نوع Background Worker واربطها بالمستودع.
2. Render سيتبع ملف `render.yaml` الموجود في المستودع ويستخدم:
   - build command: `npm ci`
   - start command: `npm start`
3. أضف المتغيرين البيئيين DISCORD_TOKEN و CLIENT_ID في إعدادات الخدمة على Render.

ملاحظات:
- الأوامر الشلّاش (Slash Commands) تُسجَّل تلقائياً عند بدء التشغيل عبر الـ REST. تسجيل الأوامر العالمية (global) قد يستغرق بعض الوقت لتظهر في جميع السيرفرات.
- إذا رغبت بتسجيل الأوامر لمجموعة اختبار فوراً (guild commands) لتظهر مباشرة، أخبرني لأقوم بتعديل التسجيل ليستخدم Routes.applicationGuildCommands مع معرف السيرفر الخاص بك.
